# PCTFFM (파일좀 변환해줘라)

PCTFFM은 **FFmpeg 기반의 파일 변환 프로그램**으로,  
비디오 / 오디오 / 이미지 파일을 다양한 포맷으로 변환할 수 있는  
Windows용 WinForms 애플리케이션입니다.

---

## ✨ 주요 기능

- 🎞 비디오 / 🎵 오디오 / 🖼 이미지 포맷 변환
- 📂 다중 파일 동시 변환 지원
- ⚙ 동시 변환 개수 설정 가능
- 🧠 FFmpeg 자동 다운로드 및 초기화
- 📊 진행률 표시 및 상태 관리
- 🗑 변환 완료 후 원본 파일 삭제 옵션
- 🖱 드래그 앤 드롭 파일 추가 지원

---
